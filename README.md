# Hackethos4u



## About the Apps
Hackethos4u is an E-Learning, Learning Management System focused on Cybersecurity and Ethical Hacking. Users can learn through videos, materials from slides and quizzes.

## Technologies
- [Flutter](https://docs.flutter.dev/get-started/install)
- [Dio](https://pub.dev/packages/dio)
- [Providers](https://pub.dev/packages/provider)
- [Flutter Youtube Player](https://pub.dev/packages/youtube_player_flutter)
- [Flutter Easy Loading](https://pub.dev/packages/flutter_easyloading)
